# Mattys World

Your personal information organizer — a modern take on 37signals' Backpack.

## Features
- 📝 **Pages** — freeform wiki-style pages with notes and to-do lists
- ✅ **To-Do Lists** — check off items, edit inline
- 📅 **Calendar** — month view with color-coded events, recurring events, reminders
- 📎 **File Attachments** — attach files to any page
- 🔗 **Page Sharing** — generate public links
- 🔔 **Reminders** — 15 min, 1 hour, or 1 day before notifications
- 🔁 **Recurring Events** — daily, weekly, monthly
- 🔒 **Auth** — email/password sign-in
